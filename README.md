## Критерії виконання

Критерії виконання Створити папку для проекту під назвою genius-homework-4. Для
стилів створити наступну структуру, розбивши CSS-файл на SCSS-файли і
директорії: Screenshot 2023-03-22 at 22 08 47

Виконати іменування класів в проекті за методологією БЕМ. Зробити адаптив
сторінки за стратегією Mobile first, використовуючи медіа-запити на трьох точках
перелому: 480px, 768px, 1200px. Всі зображення (як контентні, так і фонові)
повинні бути оптимізовані під retina-дисплеї. Створити мобільне меню і додати
функціонал його роботи, попередньо підключивши файл mobile-menu.js.
